VA Logging API

This page is the primary endpoint for recording va logging events and is not meant for public consumption. Specifically, the View for this page records certain javascript events on the front end of VA explorer by processing POST requests to this URL (va_logs/submit_log.html).

See /va_explorer/static/logging_event_handlers.js for examples of how to reach this API.

About VA Explorer

VA (Verbal Autopsy) Explorer is a prototype application, built to demonstrate verbal autopsy data management and analysis capabilities. VA Explorer currently supports the following functionality at various degrees of maturity:

Learn more »

User Management

  • Creation/disabling of user accounts by administrators
  • Password management
  • Role-based access for admins, data managers, data viewers, and field workers
  • Geographical assignment and geography-based priveleges

View details »

VA Ingest

  • Import Verbal Autopsy questionnaire responses
  • Integrate with survey tools like Open Data Kit (ODK)

COD Assignment

  • Assign cause of death using InterVA5
  • Building support for Insilico

VA Management

  • View errors and warnings from COD assignments
  • Fix errors in existing VAs
  • Track change history
  • Revert latest changes

VA Analytics

A flexible dashboard enabling quick insights. For a chosen region:

  • Inspect geospatial trends with dynamic heatmap
  • View top overall causes of death
  • Explore VA distributions by age, gender, and place of death
  • Visualize trends over time
